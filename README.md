# 🎵 YouTube QR Card

A simple and stylish QR Card built using HTML and CSS that allows users to quickly access a YouTube playlist by scanning a QR code.

---

## 🚀 Features

* 🎨 Clean and modern UI design
* 📱 Fully centered layout using Flexbox
* 🌈 Beautiful gradient background
* ✨ Smooth hover animations
* ⚡ Lightweight and fast

---

## 📸 Preview

Scan the QR code to instantly open the YouTube playlist and enjoy your favorite songs 🎧

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, Transitions, Gradients)

## 💡 How It Works

1. Open the webpage
2. Scan the QR code using your phone
3. Get redirected to the YouTube playlist instantly
