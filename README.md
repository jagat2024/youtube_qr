# 🎵 YouTube QR Card

A simple and responsive QR Card project built using HTML and CSS.
This project displays a stylish card with a QR code that links to a YouTube playlist.

## 🚀 Features

* Clean and modern UI
* Centered card layout using Flexbox
* Responsive design
* Smooth hover effects
* Custom styling with gradients

## 📸 Preview

Scan the QR code to open the YouTube playlist directly.

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Gradients)
